# NumberEntry

[img](http://my-owo-ink.b0.upaiyun.com/puge/github/module/NumberEntry.png)

数字选择控件，带增加减少按钮

# Props

| Props       | 属性          | 默认值    | 说明         |
| ----------- |:--------------| ---------|--------------|
| value    | Number string  | 无       | 组件绑定的值 |
| max      | Number  | 无       | 能输入的最大值 |
| min      | Number  | 无       | 能输入的最小值 |
| name      | String  | 无       | 显示标签文字 |
| fun      | Function  | 无       | 值发生改变的回掉函数 |
